- 方法是类中的[[函数及方法类型|函数]]成员，用于描述类的行为或操作。它们定义在类的内部，可以对类的属性进行操作。方法也是一种类型
- 方法可以接受参数，并且可以返回一个值。在方法内部，通过 `self` 参数可以访问类的属性和其他方法。
- <font color="#c0504d">实例方法（Instance Methods）</font>
	- 在实例方法中，`self` 是一个特殊的参数，用于**引用类的实例自身**。方法的定义中第一个参数为 `self`。通过 `self`，可以访问类的属性和其他方法，以及在方法内部操作实例的数据。
- <font color="#c0504d">类方法（Class Methods）</font>
	- 在类方法中，`cls` 是一个特殊的参数，用于**引用类自身**。它通常作为类方法的第一个参数传递，用于操作类属性或调用其他类方法。
- <font color="#c0504d">静态方法（Static Methods）</font>
	- 在静态方法中，不接收特定的第一个参数（如`self`或`cls`）。它是一个普通函数，属于类的命名空间，放在类里是因为它在逻辑上属于类。
